## Installation

These commands will install the latest version of MySQL.

```bash
brew install mysql
brew services start mysql
```

You can see available versions of MySQL via the following:

```bash
brew search mysql
```

Install MySQL 5.7 via the following:

```bash
brew install mysql@5.7
```
